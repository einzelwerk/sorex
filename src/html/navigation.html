Build

Build Commands:

npm run dev

Launches developer mode

npm run build

Launches project build mode into the dist folder

npm run zip

Launches project build mode into an archive

npm run ftp

Launches project build mode and sending to ftp-server

npm run page -- --page=file_name --name=page_name

Creation of a new page and all necessary files for the page to work in the project
